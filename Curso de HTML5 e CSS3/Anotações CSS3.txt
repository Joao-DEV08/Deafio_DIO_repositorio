Em 1996, foi criado o CSS, para formatar as páginas da web com estilos diferentes.

seletores: 

a, p, h1, h3 {
  color: blue;
  font-size: 14px;
}

No exemplo acima, temos os elementos que serão modificados e a modificação, no caso para cor
azul e o tamanho da fonte para 14.

Box Model:

.post {
    background-color: #fff;
    padding: 10px;
    border: 3px solid #000;
    margin: 10px;
}

O box model, é um modelo para criar uma "caixa", com bordas e cores para dar um aspecto
diferente ao projeto.
